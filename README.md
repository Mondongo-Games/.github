## Instrucciones

### Instalación:
- UNZIP
- EJECUTAR `SGI.EXE`

### CONTROLES:

| Tecla                | Acción                           |
|----------------------|----------------------------------|
| A, W, S, D            | Movimiento                       |
| E                     | Abrir cofres / Equipar arma      |
| Click IZQ/DER         | Disparo                          |
| Espacio               | Dash                             |
| ESC                   | Menú                             |
| L                     | Trampas, subir nivel             |
